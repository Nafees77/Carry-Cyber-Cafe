# **App Name**: Carry Cyber Cafe - Dynamic Web Presence

## Core Features:

- Dynamic Header: Display a sticky header with a logo and navigation links, fully managed via Firestore. Includes mobile hamburger menu.
- Responsive Banner: Show a full-width banner with different images for desktop, tablet, and mobile, configurable in Firestore.
- Services Table: Display services in a grid with English and Hindi text, loaded dynamically from Firestore.
- Contact Information: Present dynamic contact details, opening hours, and a Google Maps embed, fetched from Firestore.  Includes a contact form with Firestore integration.
- Reasons to Choose Us: Display a dynamic list of reasons (English/Hindi) loaded from Firestore using clean, simple cards.
- Customer Reviews Carousel: Show customer reviews in a dynamic slider or carousel, with data sourced from Firestore.
- Dynamic FAQ: Implement a dynamic FAQ accordion with questions and answers in English and Hindi, managed through Firestore.
- Chatbot Integration: Integrate a chatbot that saves messages to Firestore and sends user messages to WhatsApp or Telegram, based on Firestore settings.

## Style Guidelines:

- Primary color: Blue (#3B82F6) to represent trust and technology.
- Secondary color: Indigo (#6366F1) to add depth and a modern feel.
- Background color (light): Light gray (#F5F7FA) for a clean and open interface.
- Background color (dark): Dark navy (#0F172A) for footer to add richness.
- Text color (dark): Dark gray (#1E293B) for readability against light backgrounds.
- Text color (light): White (#FFFFFF) for contrast on dark backgrounds.
- Accent color: Teal (#14B8A6) for highlights and calls to action.
- Body and headline font: 'Inter', a sans-serif font providing a modern, machined, neutral look; well-suited to both headlines and body text.
- Use a set of consistent, modern icons for services, contact info, and social links.
- Maintain a clean, modern layout with sufficient spacing between elements and sections for enhanced readability.
- Use smooth, subtle animations on scrolling and when content loads to improve user experience.
